## 2026年保研关键时间轴

根据教育部及多所高校发布的政策文件（[参考文件](https://yz.chsi.com.cn/yzzt/tmzn/#bmlc))，2026年保研参考关键节点如下表所示，建议考生提前设置提醒，避免错过重要截止日期：

| 时间 | 事件 | 具体内容 |
| :--- | :--- | :--- |
| **2026年3月–4月** | 启动复习与文书准备 | 稳住绩点，启动专业课与英语复习；撰写个人陈述（PS）、简历（CV）初稿；联系推荐人 |
| **2026年5月–6月** | 夏令营集中申请 | 各高校密集发布夏令营通知，完成材料投递；部分院校开始公布入营名单 |
| **2026年7月–8月** | 参加夏令营 | 参与线下/线上夏令营活动，接受笔试、面试考核；获得“优秀营员”即获拟录取资格 |
| **2026年8月底–9月上旬** | 预推免申请冲刺 | 未获理想offer者继续申请预推免；目标院校发布预推免通知与考核 |
| **2026年9月中旬（约10-18日）** | 本校推免资格公示 | 本科院校完成推免生名单公示，学生正式获得推免资格及推免编号 |
| **2026年9月18日左右** | **系统开放：注册与填报个人信息** | 登录[全国推荐免试攻读研究生信息公开管理服务系统](https://yz.chsi.com.cn/tm)，注册账号、核实学籍学历、**填报个人资料信息、上传照片**。 |
| **2026年9月18日-22日** | **系统开放：网上支付** | 在上述时间段内，按推荐高校所在省级教育招生考试机构的规定，完成报名费网上支付。 |
| **2026年9月22日9:00起** | **志愿填报正式开放** | 推免生可**同时填报三个平行志愿**（不区分先后顺序）。每个志愿提交后48小时内不允许修改，请谨慎填报。 |
| **2026年9月23日9:00起** | **查看并确认复试通知** | 招生单位开始发送复试通知。推免生可登录系统查看，并须在**招生单位规定的时间内**通过系统确认是否参加复试。未按时回复，招生单位可取消复试通知。 |
| **2026年9月25日14:00起** | **查看并确认待录取通知** | 通过复试后，招生单位发送待录取通知。推免生可登录系统查看，并须在**招生单位规定的时间内**通过系统确认是否接受。**一旦确认接受，即正式完成录取，不能再填报其他志愿或接受其他通知。** |
| **2026年10月初至中旬** | “九推”捡漏阶段 | 尚未招满的院校释放补录名额，未获得满意offer的同学可主动联系争取最后的机会。 |

> **特别提醒：** 近年来部分顶尖高校的个别院系调整了夏令营形式或预推免时间，出现“预推免抢跑”趋势（如部分院系将预推免提前至7月），申请人应密切关注目标院校具体院系的政策变化。此外，9月底系统正式开放当天（即俗称的“开网日”），确认待录取通知的时间往往非常紧凑，请务必保持通讯畅通并随时关注系统动态。

## 保研贴常用术语释义表

以下为保研过程中常见术语的完整释义，涵盖申请、考核、录取各阶段的核心“黑话”，帮助理解圈内交流语境与隐性规则。

| 术语名称 | 解释 |
| :--- | :--- |
| **bg/背景** | 指申请者的本科院校、成绩、英语、科研等综合条件，是学术背景的简称 |
| **GPA** | Grade Point Average的缩写，指平均学分绩点，在保研语境下特指大学期间所有课程成绩的加权平均分 |
| **rank/rk** | 成绩排名的简称，至关重要，关乎能否入营和申请推免，强com好学校基本上卡双非前百分之几 |
| **92** | 指985、211学校的同学，是保研圈中的常用称呼 |
| **双非** | 指非985、211的学校，是保研圈中的常用术语 |
| **C9** | 九校联盟，包括清北、华五、西安交大、哈工大，是中国顶尖高校的代表 |
| **华五** | 指复旦大学、上海交通大学、南京大学、浙江大学、中国科学技术大学这五所顶尖高校 |
| **title/帽子** | 指学校的头衔，如985、211、双一流，有的学生追求“title upgrade”，即想去一个头衔比本科更好的学校 |
| **夏令营** | 全称“推荐免试研究生夏令营”，是高校在每年暑假举办的学术交流与人才选拔活动，通常在6-8月举办，表现优异者可获“优秀营员”资格 |
| **预推免** | 保研的第二战，集中在9月正式推免系统开放前，节奏快、效率高，通常只有笔试面试，是夏令营失利的同学“复活”的主战场 |
| **九推** | 全称“九月正式推免”，指每年9月28日国家推免服务系统开放到10月25日左右结束的正式录取环节，是保研流程的最终阶段 |
| **优营** | 即“优秀营员”的简称，是夏令营中表现优异获得的称号，绝大多数情况下等价于拿到了拟录取资格 |
| **优营有效力/铁offer** | 夏令营中获得优秀营员后，学校承诺只要推免系统开放后选择该学校就一定会被录取，具有确定性 |
| **优营无效力** | 学校超发优营，夏令营中获得优秀营员但还需要参加预推免或根据排名录取，不具有确定性 |
| **候补/wl** | Waiting List的缩写，如果前面的人当鸽子放弃优营，处于候补队列的人会按序依次递补 |
| **拟录取** | 预推免或正式推免中，院校向学生发出的预备录取通知，学生确认后即锁定该院校录取资格 |
| **待录取** | 研究生招生中的教育术语，指考生通过复试后，招生单位通过全国硕士研究生招生调剂服务系统发送的拟录取意向通知，分为正常待录取与限制性待录取两种类型，考生确认后招生单位调取档案并提交教育部审批完成正式录取 |
| **推免服务系统** | 全国统一的推免工作信息备案公开平台和推免生网上报考录取系统，所有推免生均通过此系统填写报考志愿、接收并确认招生单位的复试及待录取通知 |
| **928/929** | 保研系统开放当天，每个人只能填写最多三个志愿，相互独立，先接受复试，再接受拟录取 |
| **上岸** | 指最终确认了心仪的录取学校，保研之旅圆满结束，是保研成功的代名词 |
| **海王/渣男校** | 指某些学校发大量优营或候补，导致最终录取率极低，广撒网养鱼的行为 |
| **放鸽子** | 指学生拿到了offer但最后不去了，放学校鸽子 |
| **被鸽** | 指由院校或者导师放弃你的录取资格 |
| **全聚德/全拒德** | 调侃说法，指申请的所有项目都被拒绝，颗粒无收 |
| **套磁/陶瓷** | 提前给目标院校意向导师发邮件，介绍自己的情况，询问招生名额，争取导师的青睐，在弱com院校中提前联系导师极为重要 |
| **强com/弱com** | “com”指招生委员会，强com指录取决定权在招生委员会，套磁导师作用不大；弱com指导师话语权很大，只要导师点头，委员会只是走流程 |
| **BQ/行为面试** | 通过问你“过去经历中遇到困难怎么解决”之类的问题，考察你的行为和性格特质 |
| **case/群面/无领导** | 多见于经管、法学等专业的面试形式，给一个案例让小组讨论，观察每个人的表现 |
| **英文面/口语面** | 用英语进行的面试环节，可能包含自我介绍、日常对话或专业问题 |
| **笔面** | 即笔试和面试的合称，是保研考核中的常见形式 |
| **mock** | 即模拟面试的简称，提前熟悉面试流程，进入面试状态 |
| **CV** | 即个人简历的简称，一页纸概括申请者的教育背景、科研经历、获奖情况等 |
| **PS** | 即个人陈述的简称，是一篇讲述个人故事和学术追求的长文，解释申请者为什么适合这个项目 |
| **RL** | 即推荐信的简称，通常需要2-3封副教授以上职称的老师为申请者写推荐信 |
| **Pub** | 即论文发表的简称，指已发表的学术论文，是科研能力的有力证明 |
