欢迎浏览**统计专业档案库**！本档案库收录了各高校统计学及相关专业的保研信息，包括夏令营与预推免时间线、考核细节及真实经验贴。

请从左侧边栏或右上角选择您感兴趣的高校档案进行查阅。如需贡献新的档案，请参阅[贡献指南](../contribute)。

## 2026年保研关键时间轴

根据教育部及多所高校发布的政策文件（[参考文件](https://yz.chsi.com.cn/yzzt/tmzn/#bmlc))，2026年保研参考关键节点如下表所示，建议考生提前设置提醒，避免错过重要截止日期：

| 时间 | 事件 | 具体内容 |
| :--- | :--- | :--- |
| **2026年3月–4月** | 启动复习与文书准备 | 稳住绩点，启动专业课与英语复习；撰写个人陈述（PS）、简历（CV）初稿；联系推荐人 |
| **2026年5月–6月** | 夏令营集中申请 | 各高校密集发布夏令营通知，完成材料投递；部分院校开始公布入营名单 |
| **2026年7月–8月** | 参加夏令营 | 参与线下/线上夏令营活动，接受笔试、面试考核；获得“优秀营员”即获拟录取资格 |
| **2026年8月底–9月上旬** | 预推免申请冲刺 | 未获理想offer者继续申请预推免；目标院校发布预推免通知与考核 |
| **2026年9月中旬（约10-18日）** | 本校推免资格公示 | 本科院校完成推免生名单公示，学生正式获得推免资格及推免编号 |
| **2026年9月18日左右** | **系统开放：注册与填报个人信息** | 登录[全国推荐免试攻读研究生信息公开管理服务系统](https://yz.chsi.com.cn/tm)，注册账号、核实学籍学历、**填报个人资料信息、上传照片**。 |
| **2026年9月18日-22日** | **系统开放：网上支付** | 在上述时间段内，按推荐高校所在省级教育招生考试机构的规定，完成报名费网上支付。 |
| **2026年9月22日9:00起** | **志愿填报正式开放** | 推免生可**同时填报三个平行志愿**（不区分先后顺序）。每个志愿提交后48小时内不允许修改，请谨慎填报。 |
| **2026年9月23日9:00起** | **查看并确认复试通知** | 招生单位开始发送复试通知。推免生可登录系统查看，并须在**招生单位规定的时间内**通过系统确认是否参加复试。未按时回复，招生单位可取消复试通知。 |
| **2026年9月25日14:00起** | **查看并确认待录取通知** | 通过复试后，招生单位发送待录取通知。推免生可登录系统查看，并须在**招生单位规定的时间内**通过系统确认是否接受。**一旦确认接受，即正式完成录取，不能再填报其他志愿或接受其他通知。** |
| **2026年10月初至中旬** | “九推”捡漏阶段 | 尚未招满的院校释放补录名额，未获得满意offer的同学可主动联系争取最后的机会。 |

> **特别提醒：** 近年来部分顶尖高校的个别院系调整了夏令营形式或预推免时间，出现“预推免抢跑”趋势（如部分院系将预推免提前至7月），申请人应密切关注目标院校具体院系的政策变化。此外，9月底系统正式开放当天（即俗称的“开网日”），确认待录取通知的时间往往非常紧凑，请务必保持通讯畅通并随时关注系统动态。
