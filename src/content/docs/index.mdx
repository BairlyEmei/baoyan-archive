**保研信息开源档案库（统计学专版）** 是一个由保研人共同维护的开源数据库，系统收录并结构化整理了统计学专业的保研信息。

我们收录各高校统计学及相关专业的 **夏令营 / 预推免时间线**、**考核细节**（笔试范围、面试偏好、背景门槛）以及来自小红书、知乎等平台的 **真实经验帖**，全部开源、可持续维护。

</div>

## ✨ 平台特色

<CardGrid stagger>
  <Card title="专业数据模型" icon="document">
    标准化的统计学保研信息格式，全面覆盖考核要求、时间线、招生规模等关键信息，让信息一目了然。
  </Card>
  <Card title="AI 辅助贡献" icon="pencil">
    引入「AI 数据提取 + 可视化表单微调」工作流，无需懂代码，四步即可完成一份高质量档案提交。
  </Card>
  <Card title="完全开源透明" icon="github">
    所有数据托管在 <a href="https://github.com/BairlyEmei/baoyan-archive" target="_blank" rel="noopener noreferrer">GitHub</a>，版本可溯源，信息可验证。欢迎通过 Issue 指出错误或提交 PR 完善数据。
  </Card>
  <Card title="社区持续维护" icon="star">
    随招生季动态更新，历史数据可追溯，当前数据由社区成员共同校验。
  </Card>
</CardGrid>
